# Subtractor
A sequence to sequence subtractor by LSTM

## Prerequisite
- Python 3.6.4

## Usage

Basic
```
pip install
jupyter notebook
```

Run Different Training Size & Digits
```
python main.py
```

Direct Load Model & Testing
```
python model.py
```
